# Currency Converter

A simple web application to convert an amount from one currency to another using a free currency API. It also dynamically updates the country flags for selected currencies.

## Features
- Convert any amount between two currencies.
- Real-time conversion using [Fawaz Ahmed Currency API](https://github.com/fawazahmed0/currency-api).
- Dynamic country flags update according to the selected currencies.
- Input validation (default amount set to 1 if left empty or invalid).

## Technologies Used
- HTML
- CSS
- JavaScript
- Fetch API
